## `setTimeOut` 和 `setInterval`

`setTimeOut` 延迟几秒钟后再执行代码：

```javascript
console.log('A');
setTimeout(() => {console.log('B');}, 1000);  // 延迟 1000 毫秒
console.log('C');
```

结果是
```text
A
C
B
```

而且 B 是过一会才出现的。

`setInterval` 很像，只不过在第一个延迟执行之后，每隔一会儿还会执行以下。
